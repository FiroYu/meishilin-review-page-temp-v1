# 携程美食林 · 2027 年度专家理事评审页（Trip.Gourmet）

美食林（英文名 **Trip.Gourmet**）2027 年度高端榜单**专家理事评审 H5 投票页**的临时预览站。
单文件 `index.html`，仅依赖两个 CDN（Tailwind + lucide），开箱即用。

> 🔗 在线预览：https://firoyu.github.io/meishilin-review-page-temp-v1/

---

## 设计规范

按《极简低保真设计规范》三条基因法则构建：

1. **纯灰度 Tailwind Gray 色阶** — 全站无彩色，唯一例外是校验失败的 `#DC2626`（红）。
2. **4pt 空间网格** — 圆角统一 4 / 8 / 12 / 16 / full。
3. **几何抽象占位** — 缩略图与徽标用 `bg-gray-200` + lucide 图标，不用首字母或渐变。

### 评级档位 · 三档灰度视觉编码

餐厅列表按等级**降序**排列，并通过三档亮度梯度一眼可分（亮度越高 = 等级越低，色盲友好）：

| 档位 | 英文 | 视觉编码 | 等级 |
|------|------|----------|------|
| 黑钻 | Black Diamond | 黑底白字 `#111827` | 最高 |
| 钻石 | Diamond | 中灰底白字 `#9ca3af` | 中 |
| 铂金 | Platinum | 白底灰字 + 描边 | 最低 |

## 功能流程

身份填写 → 城市选择（亚 / 欧 / 北美，国家 · 城市级联 + 搜索）→ 餐厅评审 → 增补候选餐厅 → 提交前复核 → 评审存根。

- **评审**：保持现有评级 / 建议调整评级，附评定依据（≥15 字）。
- **草稿保护**：填写中的评定依据写入 `sessionStorage`，误关抽屉不丢失，重开自动恢复。
- **已评审卡片**：按钮转为「查看 / 修改」弱化态，区别于待评审的实心主按钮。
- **提交前复核**：已评（保持 + 调整）汇总 + 增补候选审计，支持模拟下一位评审重置。
- **中英双语**：全量 i18n，一键切换，含术语专业化（City / Rationale / Reference No. 等）。
